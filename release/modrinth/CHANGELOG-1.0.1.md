# GuildMark 1.0.1

First public open-source release of GuildMark.

## Highlights

- Create and manage local guilds and player assignments through the in-game editor.
- Load guild marks from local PNG, JPG and WebP files, the clipboard, or HTTPS URLs.
- Display marks on capes, chest armor, shields and elytra.
- Mark your guild and allied players with configurable head colors.
- Browse guild members as small 3D player models in the map view.
- Import a complete remote guild database and its images with Auto Import.
- Export and import local guild data as JSON.
- Choose English or Polish and limit downloaded banner resolution.
- Support for Minecraft 1.21.8 and 26.1.2 with Fabric API.

## Fixes

- Head markers now match the exact vanilla player-head dimensions and no longer extend outside the head bounds.
- Configuration files use atomic replacement where the operating system supports it.

GuildMark is client-side only and is now available under the Apache License 2.0.
